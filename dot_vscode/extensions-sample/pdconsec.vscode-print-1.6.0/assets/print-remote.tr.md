1. Rozete tıklayarak Print'i uzak ana bilgisayara yükleyin (resme bakın) 
2. Yazdırmayı normal şekilde kullanın

Birçok Markdown uzantısı uzak ana bilgisayarda çalışacak şekilde yapılandırılmamıştır. Bu, uzak bir çalışma alanından Markdown yazdırırken bunları kullanılamaz hale getirir. Bir Markdown uzantısının yerel olarak çalıştığını ancak uzaktan çalışmadığını fark ederseniz, yazardan bu üst düzey anahtarı eklemesini isteyin

"extensionKind": ["çalışma alanı"],

Sorunlu uzantının 'package.json' dosyasına.
